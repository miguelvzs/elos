O Projeto

Sistema web desenvolvido como atividade avaliativa em Desenvolvimento Web, curso de Análise e Desenvolvimento de Sistemas. Dados armazenados via LocalStorage, sem necessidade de servidor.

Tecnologias

  • CSS3
  • JavaScript puro — map, filter, reduce
  • Leaflet.js + OpenStreetMap
  • API ViaCEP
